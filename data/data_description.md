
**sight** (景点数据集)

- .csv
- .json

| 字段名 | 含义 |
| --- | --- |
| sid | 景点id |
| name | 景点名称 |
| degree | 景点等级 |
| city | 所属城市 |
| district | 下级城市 |
| address | 详细地址 |
| location | 经纬度 |
| phone | 联系电话 |
| website | 景点网页 |
| description | 景点描述（江门景点均为空） |
